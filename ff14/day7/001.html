Local Area Network

  1. 저장장치 공유
  2. 프린터 공유

인터넷의 등장

  • LAN들을 연결
  • 한쪽선에 문제가 생겨도 우회해서 통신이 가능
  • router : 데이터의 목적지가 우리면 흡수, 아니면 목적지 쪽으로 패스

인터넷과 서비스

인터넷을 위한 통신규약 : TCP/IP

  • 이메일 서비스 : sntp 프로토콜
  • www 서비스 : http 프로토콜
  • 파일 다운로드 : ftp 프로토콜

internet, intranet, extranet

  • intranet : TCP/IP를 사용하는 LAN
  • ectranet : 인트라넷을 암호화 장비로 연결
